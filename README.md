# SQLAnalyst.ai


AI-Powered Automated Sales Intelligence Platform

SQLAnalyst.ai is an end-to-end automated analytics system that transforms raw SQL sales data into **actionable executive insights delivered every morning to managers**.

The system automatically extracts KPIs, analyzes sales performance using AI, and distributes a **daily executive report via email**, while also providing **real-time monitoring dashboards and business intelligence dashboards**.

---

# Project Objective

Managers often spend time manually checking dashboards, running SQL queries, and preparing sales reports.

SQLAnalyst.ai automates this process by:

• Extracting daily sales KPIs  
• Generating AI-driven business insights  
• Delivering an executive sales briefing every morning  
• Providing real-time monitoring dashboards  

The goal is to enable **faster data-driven decision making.**

---

# Key Features

## Automated KPI Extraction

The system queries a SQL sales database to compute:

• Daily Revenue  
• Previous Day Revenue  
• Revenue Change Percentage  
• Total Orders  
• Top 3 Revenue Generating Products  

These KPIs are calculated using optimized SQL queries.

---

## AI Generated Executive Insights

A Large Language Model analyzes KPI data and produces structured insights including:

• Executive Summary  
• Revenue Trend Analysis  
• Product Contribution Insight  
• Business Risk Signal  
• Strategic Recommendation  

This converts raw metrics into **decision-ready intelligence.**

---

## Automated Morning Email Report

Every morning at **7:00 AM**, the system automatically:

1. Extracts sales KPIs from the database  
2. Generates AI insights  
3. Formats results into an executive report  
4. Sends the report via email to managers  

Managers receive a **clear snapshot of business performance before the workday begins.**
